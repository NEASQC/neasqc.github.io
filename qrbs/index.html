Welcome to the Quantum Rule-Based System library documentation

This is the documentation page for the Quantum Rule-Based System (QRBS) software library of NEASQC project (WP6).

If this is your first time around here, you might want to check out the installation guide.

We also have some examples of usage of the library, as well as the documentation of all the elements in each module.
