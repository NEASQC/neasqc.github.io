QQuantLib.finance.classical_finance.bs_density(s_t: float, s_0: float, risk_free_rate: float, volatility: float, maturity: float, **kwargs)ÔÉÅ

Evaluates the Black-Scholes density function at s_t for a given set of parameters. The formula is:

Notes

\[\dfrac{1}{S_T\sigma\sqrt{2\pi T}}\exp\left(-\dfrac{ \ \left(\log(S_T)-\mu\right)}{2\sigma^2T}\right)\]

where \(\mu = (risk\_free\_rate-0.5\sigma)T+\log(S_0)\).

Parameters:

  • s_t (float) ‚Äì point where we do the evaluation

  • s_0 (float) ‚Äì current price

  • risk_free_rate (float) ‚Äì risk free rate

  • volatility (float) ‚Äì the volatility

  • maturity (float) ‚Äì the maturity

Returns:

density ‚Äì value of the Black-Scholes density function in s_t

Return type:

float

QQuantLib.finance.classical_finance.bs_sde_solution(x_: numpy.array, s_0: float, risk_free_rate: float, volatility: float, maturity: float, **kwargs)ÔÉÅ

For a certain parametrization \(x\) it returns a value of the underlying \(S_T(x)\) and the probability density of that value of the underlying.

Notes

The formulas are:

\[S_T = S_0e^{\sigma x+(risk_free_rate-\sigma^2/2)t} p(S_T(x)) = N(x;mean = 0, variance = T)\]
Parameters:

  • x (numpy array) ‚Äì parametrization

  • s_0 (float) ‚Äì current price of the underlying

  • risk_free_rate (float) ‚Äì risk free rate

  • volatility (float) ‚Äì the volatility

  • maturity (float) ‚Äì the maturity

Returns:

  • s_t (numpy array) ‚Äì value of the underlying corresponding to parameter x

  • probability_density (numpy array) ‚Äì probability density corresponding to s_t
